ByteBanter is a Burp Suite extension that uses BurpAI to generate context-aware payloads for Burp Intruder. It registers as a custom payload generator, supplying dynamically created payloads based on user-defined prompts during Intruder attacks.

Features

  • Generates payloads dynamically using BurpAI, guided by user-defined prompts describing the target testing scenario.
  • Registers as a custom payload generator within Intruder, selectable via the Extension-generated payload type.
  • Allows users to write and customize prompts that instruct the AI model on what payloads to produce.
  • Optionally tracks target HTTP responses and extracts content using a configurable regular expression to inform payload generation.
  • Includes a prompt optimization feature that asks BurpAI to refine a user-supplied prompt automatically.
  • Persists all settings automatically across sessions.

Usage

  1. After loading the extension, navigate to the ByteBanter tab that appears in the main interface.
  2. Write a prompt in the provided field to instruct the AI model on the type of payloads to generate.
  3. Optionally enable response tracking and enter a regular expression to extract relevant content from HTTP response bodies.
  4. Optionally click the Optimize! button to have BurpAI refine your prompt.
  5. Open the Intruder tab, configure your target and payload positions as usual.
  6. In the Intruder Payloads tab, set the payload type to Extension-generated, then select ByteBanter from the list of available generators.
  7. Start the Intruder attack. ByteBanter will supply AI-generated payloads dynamically for each request.
